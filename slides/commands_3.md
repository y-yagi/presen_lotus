
## Commands

```
bundle exec lotus --help
Commands:
  lotus console          # starts a lotus console
  lotus db [SUBCOMMAND]  # manage set of DB operations
  lotus generate         # generates app, action, model, mailer or migration
  lotus help [COMMAND]   # Describe available commands or one specific command
  lotus new              # generates a new application
  lotus routes           # prints routes
  lotus server           # starts a lotus server
  lotus version          # prints Lotus version
```
